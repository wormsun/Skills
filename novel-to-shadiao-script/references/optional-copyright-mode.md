# Optional Copyright Mode

Use this reference only when the user explicitly asks for 原创化, 低相似, 避免版权风险, 改名换设定, or similar.

Default behavior is faithful adaptation. Do not activate this mode silently.

Never auto-activate this mode. If the user asks for risk reduction but does not specify strength, confirm whether they want light, medium, or heavy transformation before applying it.

## Low-Similarity Rewrite

Reduce similarity by changing:

- Names and titles.
- Worldbuilding labels.
- Relationship labels.
- Scene location and props.
- Order of non-essential events.
- Conflict wrapper.
- Dialogue wording.
- Signature visual moments.

Preserve:

- Emotional function.
- Broad genre promise.
- Viewer satisfaction curve.
- Episode-level conflict and payoff.

## Transformation Levels

### Light

- Change wording and script form.
- Keep plot, roles, and major events.

### Medium

- Change names, scene dressing, and some event order.
- Keep emotional arc and core conflict.

### Heavy

- Replace setting and character identities.
- Keep only the premise pattern and emotional structure.

Ask the user which level they want if the requested risk reduction is important and unclear.
