# Shadiao Comedy Style

The goal is funny, fast, and clear. Comedy must serve the story.

## Core Techniques

- Contrast: serious situation plus ridiculous response.
- Overreaction: crowd reaction bigger than the event.
- Deadpan narration: narrator calmly explains absurdity.
- Status reversal: weak person suddenly dominates.
- Misunderstanding: characters interpret the same event differently.
- Repetition with escalation: the same gag returns stronger.

## Narration

Use narration to compress plot and add jokes:

- Keep narration punchy.
- Prefer active lines over abstract explanation.
- Add commentary after action, not before every action.
- Use one joke per beat; do not bury the plot under jokes.

Good pattern:

```text
旁白：三年前，他被全城嘲笑是废物。三年后，他回来了。还是被嘲笑。区别是，这次他带了账单。
```

## Dialogue

Give each role a function:

- Protagonist: calm, sharp, occasionally deadpan.
- Antagonist: arrogant, loud, confidently wrong.
- Sidekick: panic, explain, or amplify jokes.
- Crowd: chorus reaction, short and exaggerated.
- System/narrator: precise, dry, or absurdly formal.

Keep lines short enough for animation timing. Avoid long literary speeches.

## Meme Use

- Use current internet flavor but avoid making the whole script depend on a single meme.
- Prefer durable joke structures over buzzwords.
- Do not overuse emoji, romanized slang, or platform-specific catchphrases.
- If the source is serious, keep emotional beats intact and put jokes around them.

## Visual Comedy

Add production-friendly visual beats:

- freeze frame
- exaggerated zoom
- character petrified
- crowd jaw-drop
- smoke, thunder, slapstick impact
- floating caption
- background character fainting

## Avoid

- Making every character speak in the same sarcastic voice.
- Explaining the joke after the joke.
- Turning the script into pure parody with no plot.
- Adding too many scene changes that slow production.
