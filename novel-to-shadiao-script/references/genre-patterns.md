# Genre Patterns

Use genre conventions to sharpen conflict, jokes, and hooks.

## 玄幻/修仙

- Core beats: 受辱、隐藏天赋、长老施压、宗门规则、境界突破、神器、妖兽、公开排名。
- Comedy: 严肃修炼术语搭配日常反应，长老夸张失态，主角冷静吐槽。
- Hook examples: "全宗门都以为他是废柴，结果测灵石当场裂成了两半。"

## 都市/赘婿

- Core beats: 被看不起、家庭施压、职场羞辱、隐藏身份、财富或实力揭露、当众打脸。
- Comedy: 亲戚团充当合唱队，势利眼夸张表演，主角越冷静旁人越崩溃。
- Hook examples: "丈母娘刚把他赶出门，门口三十辆豪车排队喊少爷。"

## 霸总/甜宠

- Core beats: 误会、契约、公开尴尬、救场、吃醋、隐藏身份。
- Comedy: 霸总台词太严肃，旁白疯狂拆台。
- Hook examples: "他冷冷甩出一张黑卡，结果收银员说余额不足。"

## 系统流

- Core beats: 系统绑定、离谱任务、惩罚、奖励、数值暴涨、任务误解。
- Comedy: 系统吐槽、任务说明像客服、奖励和场景形成反差。
- Hook examples: "系统让他三分钟内装成高手，可他连剑都拿反了。"

## 末世

- Core beats: 危机爆发、资源争夺、背叛、异能觉醒、基地规则、怪物追击。
- Comedy: 严肃灾难中加入荒诞生存细节，主角过分淡定。
- Hook examples: "丧尸冲进超市时，他第一反应竟然是先抢打折鸡腿。"

## 悬疑/灵异

- Core beats: 异常现象、线索、误导、反转、隐藏身份、禁忌。
- Comedy: 恐怖气氛被旁白拆台，角色嘴硬但身体诚实。
- Hook examples: "半夜镜子里多出一个人，主角第一句是：你交房租了吗？"

## 校园

- Core beats: 考试、社团、校霸、暗恋、误会、比赛、公开反转。
- Comedy: 学生小动作、老师突然出现、夸张班级反应。
- Hook examples: "全班等着看他零分，结果试卷答案把老师看沉默了。"

## 历史/穿越

- Core beats: 身份落差、现代知识、朝堂压力、生存危机、发明、权谋。
- Comedy: 现代常识撞古代规矩，主角用离谱解释糊弄权贵。
- Hook examples: "他刚穿越成死囚，下一秒皇帝让他现场救国。"
