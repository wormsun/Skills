# GitHub Copilot Instructions

This repository includes the reusable Skill `skills/novel-to-shadiao-script/`.

For tasks involving 小说改编, 沙雕动画剧本, 沙雕动画小助手, 分集短视频脚本, 分镜表, or Chinese fiction-to-animation scripts, follow `skills/novel-to-shadiao-script/SKILL.md`.

Keep the default assumptions:

- Chinese output.
- 沙雕动画小助手 production format.
- Audio-first `序号 | 旁白/对白 | 音效/BGM` scripts by default.
- Do not include `画面配合` unless explicitly requested for storyboard/visual handoff.
- Plot information, actions, objects, scene changes, and causality must be carried by narration/dialogue.
- Short-video platform pacing.
- No default vertical-video assumption.
- Faithful adaptation unless the user explicitly requests low-similarity rewriting or copyright-risk reduction.
- Fully check language, actions, objects, character status, scene transitions, and plot continuity/completeness before finalizing.
