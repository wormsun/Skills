# Novel To Shadiao Script

When editing or generating files related to novel adaptation, 沙雕动画剧本, 沙雕动画小助手, 分集脚本, 分镜表, or short-video fiction scripts, follow `skills/novel-to-shadiao-script/SKILL.md`.

Required output qualities:

- Opening hook within 3-5 seconds.
- Audio-first narration/dialogue that can be understood without watching the screen.
- Default production table: `序号 | 旁白/对白 | 音效/BGM`; do not include `画面配合` unless explicitly requested for storyboard/visual handoff.
- Actions, objects, scene changes, character relationships, and plot causality must be spoken in narration/dialogue.
- Short-video rhythm for 抖音, 快手, B站, 视频号, without assuming vertical export.
- Faithful adaptation by default; use low-similarity rewrite only when explicitly requested.
- Full continuity check after drafting: language, actions, objects, character status, scene transitions, and plot completeness.
